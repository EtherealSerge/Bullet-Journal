Settings & Data Portability

Sync & Backups

Since Aether runs local-first, all data remains securely in your browser. You can back it up or move it between devices below.

Google Drive Sync

Sync your bullet journal logs automatically across devices. Files are saved in your private Google Drive space.

Status: Disconnected
  1. Go to the Google Cloud Console.
  2. Create a new project (e.g. "Aether Journal").
  3. Search for "APIs & Services", go to Credentials, and configure the OAuth consent screen. Set User Type to External or Internal.
  4. Add the scope https://www.googleapis.com/auth/drive.file.
  5. Under Credentials, click Create Credentials → OAuth client ID. Select application type Web application.
  6. Under Authorized JavaScript origins, add: http://localhost:8000.
  7. Under Authorized redirect URIs, add: http://localhost:8000/index.html (exactly). If hosted online, add your page URL (e.g., https://yourusername.github.io/bullet-journal/index.html).
  8. Copy the generated Client ID and paste it above.

Reset Local Data

Delete all journal logs from local storage. If Google Drive is connected, your next sync will pull your cloud logs back down. To delete cloud logs, delete the aether_journal_backup.json file in your Google Drive.
